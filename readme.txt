～～操作説明～～
メニュー画面ではどのキーでも押すと次の画面に移動します。

プレイする際にはキーボードやマウスの入力をすべて受け付けるので、時間制限以内にボタンを押しまくってください。

リザルト画面ではスペースキーを押すことでリスタートすることができます。

全画面共通でエスケープキーを押すとゲームが終了するようになっています。
